# Frentes activos para revisión transversal

Este archivo resume frentes de trabajo para que `personal-xh` proponga mejoras. Es contexto operativo, no autoridad final.

## Regla de autoridad

El Codex orquestador de la Mac de trabajo decide que se incorpora. `personal-xh` propone, audita y prioriza.

## Seguridad

- No tocar credenciales.
- No enviar mensajes externos.
- No publicar, comprar, reservar ni contactar terceros.
- No tocar contenido operativo de ObraCash.
- Datos clinicos reales solo si el orquestador los empaqueta expresamente; si no, trabajar con reglas, casos sinteticos y contexto sanitizado.
