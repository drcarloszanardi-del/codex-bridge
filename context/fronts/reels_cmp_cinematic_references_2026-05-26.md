# Reels CMP - referencias cinematograficas para rehacer pieza rechazada

Fecha: 2026-05-26
Estado: direccion editorial, no render.

## Regla principal

No copiar activos, marcas, textos ni material de terceros. Copiar solo la gramatica audiovisual:
ritmo, relacion imagen/mensaje, densidad de texto, tipos de transicion, orden narrativo y nivel de
terminacion.

## Feedback del Doctor que queda como restriccion

- El reel rechazado tiene demasiado texto descriptivo sobre el film.
- Las transiciones no son premium.
- No alcanza con una secuencia de fotos o placas.
- La pieza debe sentirse mas cinematografica: imagen real, respiracion visual, texto minimo y cortes con sentido.

## Nueva gramatica para el reel

### Densidad de texto maxima

- 0-3 s: una frase gancho, maximo 5 palabras.
- Cada bloque: maximo 3-5 palabras.
- No usar subtitulo descriptivo debajo de titulo grande.
- Si la frase necesita mas de una linea para explicar el plano, hay que cambiar el plano o la voz.

### Transiciones permitidas

- Match cut por forma: RM / campo / logo / equipo con movimiento similar.
- Match-on-action: mano, instrumento, monitor o paso del procedimiento.
- J-cut/L-cut si hay voz o audio ambiente: el audio entra antes que la imagen.
- Push-in / pull-out lento para revelar contexto.
- Disolvencias muy breves solo para paso temporal o cambio emocional.

### Transiciones prohibidas

- Zoom de foto repetido sin finalidad narrativa.
- Cortes duros entre placas sin movimiento.
- Plantillas genericas tipo slideshow.
- Texto que entra/sale con animacion llamativa mientras la imagen medica pierde jerarquia.

## Estructura base para rehacer

Duracion objetivo: 40-55 s.

1. 0-3 s: imagen potente + idea unica.
2. 3-10 s: problema real, con RM o material propio seguro.
3. 10-22 s: estrategia, con instrumental/monitor/planificacion.
4. 22-36 s: gesto tecnico o momento quirurgico, sin exceso de texto.
5. 36-45 s: recuperacion/resultado humano si esta autorizado.
6. 45-55 s: cierre CMP sobrio, contacto legible.

## Pedido minimo de material antes de renderizar otra version

- Video real principal: 20-40 s, vertical si existe, del tema clinico.
- 3 a 5 clips B-roll propios: RM, manos, instrumental, monitor, consultorio o quirofano.
- Confirmacion del mensaje clinico en una frase.
- Confirmacion de que datos/paciente estan autorizados o deben ser difuminados.

## Gate antes de enviar al Doctor

- Contact sheet 10-12 frames.
- Vista en 540x960 para legibilidad.
- Texto en pantalla contado y justificado.
- Chequeo de privacidad cuadro a cuadro.
- Una frase de CEO: "Este reel transmite X usando estos planos por Y".
- Si esa frase suena generica, no se envia.
