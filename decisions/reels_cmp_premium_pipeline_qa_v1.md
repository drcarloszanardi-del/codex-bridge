# Reels CMP premium pipeline QA v1

## pipeline

1. Intake: fotos/videos, objetivo, audiencia, canal, duracion.
2. Contexto de album completo: seleccionar narrativa, no pieza aislada.
3. Guion: hook, valor medico, advertencia, CTA prudente.
4. Storyboard: planos, texto en pantalla, ritmo.
5. Estetica CMP: limpia, medica, premium, sin saturacion.
6. QA medico: no prometer resultados, no diagnosticar por reel.
7. QA datos publicos: telefono `2364384321`, IG `@drcarloszanardi`, web `www.centromedicopellegrini.com.ar`.
8. QA licencias: musica, imagenes, stock, derechos.
9. Prepublicacion: aprobacion humana.

## herramientas

- Gemini: analizar album/video largo.
- Nano Banana: generar assets visuales no sensibles.
- Codex: guion, checklist, caption.
- HeyGen: solo con guion aprobado.
